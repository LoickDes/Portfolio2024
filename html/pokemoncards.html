Pokemon Cards

TypeScript Tailwind ReactJS Next Api

L'objectif du projet était de développer une application web dynamique utilisant ReactJS et Next.js, permettant aux utilisateurs de consulter un catalogue de cartes Pokémon. Les données des cartes sont récupérées en temps réel via une API externe, offrant ainsi un accès fluide et interactif aux informations détaillées de chaque carte (nom, type, rareté, illustration, etc.).
