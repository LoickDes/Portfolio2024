Bugtracker

html css php sql

L’objectif pour ce projet était de développer une application de type « bugtracker » simplifiée permettant de soumettre des tickets pour une intervention concernant les bugs de fonctionnement d’un programme ou d’un jeu. De plus le Bug tracker est géré par une base de données.

Le Testeur déclare les bugs rencontrés lors de l’usage du programme ou du jeu et a accès à l’évolution du traitement de ses demandes.

Le Développeur prend en charge le bug qui lui est attribué et met à jour l’avancement du traitement jusqu’à sa clôture.

L'Admin reçoit une notification par mail du dépôt d’un bug, attribue le bug à un développeur et accès à toutes les informations concernant tous les bugs.
